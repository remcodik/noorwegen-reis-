# Noorwegen Reisplanner

Statische GitHub Pages-site. Upload **index.html** en **README.md** los in je repository, niet als zip.

## Publiceren op GitHub Pages
1. Ga naar je repository op GitHub.
2. Verwijder eventueel oude ZIP-bestanden uit de repository.
3. Upload `index.html` en `README.md` in de root van de repository.
4. Ga naar **Settings → Pages**.
5. Kies **Deploy from a branch**.
6. Branch: `main`, folder: `/root`.
7. Save.

Daarna staat de site meestal binnen enkele minuten online.
